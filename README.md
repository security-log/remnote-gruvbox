# Gruvbox for RemNote

Retro groove colors for your notes — dark and light variants with a configurable accent color.

## Variants

| Variant | Description |
|---|---|
| Dark | Standard Gruvbox dark |
| Dark Hard | Darker background |
| Dark Soft | Softer dark background |
| Light | Standard Gruvbox light |
| Light Hard | High contrast light |
| Light Soft | Softer light background |

## Accent colors

Yellow (default), Orange, Red, Green, Blue (Aqua), Purple, Aqua

## Development

```sh
npm install
npm start        # dev server on :8080
npm run build    # produces dist.zip for upload
```

Load the plugin in RemNote via **Settings → Plugins → Load Unpacked Plugin** and point it at the `dist/` folder, or upload `dist.zip`.
